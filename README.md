# zenity.cr

[Zenity](https://github.com/ncruces/zenity) wrapper for Crystal

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     zenity:
       github: your-github-user/zenity
   ```

2. Run `shards install`

## Usage

```crystal
require "zenity"
```
